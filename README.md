# **Feynman-Obsidian-LLM-Wiki**

✨ 基于 Karpathy 知识库理念 + 费曼学习法 + Obsidian 的全自动个人学习系统

## 项目介绍

本项目严格遵循 Andrej Karpathy 提出的 LLM 知识库核心思想，以 **raw - wiki - Atlas** 三层架构为基础，构建可累积、可迭代、可检索的结构化知识体系，结合费曼学习法打造全流程自动化学习闭环，彻底解决“知识碎片化、学习无反馈、笔记难沉淀”的痛点。

系统采用纯本地文件管理，无复杂依赖，不绑定任何语言、学科或学习场景，可灵活适配编程、数学、英语、职业考证、专业知识等各类学习需求，搭配 Obsidian 实现知识可视化与快速检索，是终身学习者的高效知识库工具。

## 核心特性

- ✅ **Karpathy 标准架构**：严格遵循 raw（原始资料）、wiki（结构化知识）、Atlas（全局索引）三层结构，确保知识可沉淀、可迭代

- ✅ **全自动知识构建**：一句话指令即可 ingest 权威资料，自动生成规范、详细的可学习 wiki，无需手动整理

- ✅ **费曼交互式学习**：按小节拆分知识点，讲解后引导费曼输出，AI 自动检查、评级、纠正，确保真正掌握

- ✅ **闭环学习机制**：内置阶段考试、错题解析、薄弱点补学功能，形成“学习-检验-强化”完整闭环

- ✅ **Atlas 全局导航**：全局索引与知识地图，快速定位、检索知识点，解决知识碎片化问题

- ✅ **学习笔记自动归档**：费曼输出、考试错题、补学记录自动存入 Feynman 目录，永久沉淀个人理解

- ✅ **全场景通用**：不绑定任何学科/语言，适配编程、考证、语言、专业知识等各类学习需求

- ✅ **开箱即用**：纯 Obsidian 原生支持，无需额外安装插件，本地文件安全可掌控

## 目录结构

```bash
├── raw/            # 原始权威资料（只读、不可修改，保留原文完整性）
├── wiki/           # 结构化、可学习的知识文章（AI 维护，含定义/示例/易错点）
├── Atlas/          # 全局导航、主题索引与知识地图（快速定位知识点）
├── Feynman/        # 费曼输出、学习笔记、考试与错题归档（按主题分类）
├── Skills/         # AI 自动化学习技能（可直接导入 Trae 等工具，开箱即用）
│   ├── 01_Karpathy_知识库管理员.md
│   ├── 02_交互式费曼学习导师.md
│   └── 03_技能使用说明.md
├── references/     # 统一模板目录（wiki 文章、费曼输出、索引模板）
└──  README.md       # 项目说明文档（当前文档）

```

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/你的用户名/Karpathy-Learning-System.git
```

### 2. 打开项目

直接使用 **Obsidian** 打开仓库根目录，系统会自动识别目录结构，无需额外配置。

### 3. 导入 AI 技能

1. 打开 Trae（或同类 AI 工具），进入「设置 → 规则和技能」

2. 新建技能，分别复制 `Skills/` 目录下的 3 个技能文件内容，粘贴并保存

3. 技能导入完成后，即可触发自动化学习流程

### 4. 启动学习

针对任意学习主题，输入以下指令，即可开启全自动学习：

```markdown
# 第一步：生成知识（ingest 权威资料，生成 wiki）
/Karpathy LLM 知识库管理员
ingest 【你的学习主题】权威资料，生成详细可学习 wiki，包含定义、示例、易错点，同步 Atlas 索引

# 第二步：开始交互式学习
/交互式费曼学习导师
开始学习 Wiki/【主题目录】/【对应 wiki 文件】.md
```

## 全自动学习闭环流程

1. **知识构建**：执行 ingest 指令，AI 自动抓取/录入权威原始资料，保存至 raw 目录，并生成结构化、可学习的 wiki 文章，同步更新 Atlas 全局索引。

2. **交互式学习**：AI 按小节拆分 wiki 知识点，逐节讲解，每讲完一小节引导用户进行费曼输出（用自己的话复述知识）。

3. **输出检验**：AI 对照 wiki 原文，逐点检查用户的费曼输出，指出错误、遗漏或模糊点，给出「完全掌握✅」「部分掌握⚠️」「未掌握❌」的评级，并引导修正。

4. **阶段考试**：当用户学完整个章节且所有小节达到「完全掌握」，AI 自动生成阶段考试卷（含选择、判断、主观/实操题），用户作答后自动批改。

5. **薄弱点补学**：针对考试中的错题，AI 汇总薄弱知识点，生成补学内容，引导用户重新学习、输出，直至完全掌握。

6. **持续进阶**：补学通过后，自动进入下一章节，重复上述流程，逐步构建完整的知识体系。

## 使用注意事项

- raw 目录为原始资料目录，建议保持只读，不手动修改，确保知识源头可追溯。

- 导入 AI 技能时，请完整复制 Skills 目录下的文件内容，避免遗漏规则导致流程异常。

- 费曼输出建议严格遵循 references 目录下的模板，确保内容结构化，便于 AI 检查和后续复习。

- Obsidian 打开后，可开启「关系图谱」功能，直观查看知识点之间的关联，提升学习效率。

- 若需切换学习主题，直接执行新主题的 ingest 指令即可，不影响已有知识的沉淀。
