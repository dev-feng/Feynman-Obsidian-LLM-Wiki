# Excel 函数参考 - 权威指南

## 函数类别

### 财务函数
- **FV**：计算投资的未来值
- **PV**：计算投资的现值
- **PMT**：计算贷款的每期付款额
- **NPV**：计算一系列现金流的净现值
- **IRR**：计算内部收益率

### 日期和时间函数
- **TODAY**：返回当前日期
- **NOW**：返回当前日期和时间
- **DATE**：根据年、月、日创建日期
- **YEAR**：提取日期的年份
- **MONTH**：提取日期的月份
- **DAY**：提取日期的日
- **HOUR**：提取时间的小时
- **MINUTE**：提取时间的分钟
- **SECOND**：提取时间的秒

### 数学和三角函数
- **SUM**：计算数值的总和
- **AVERAGE**：计算数值的平均值
- **MAX**：返回最大值
- **MIN**：返回最小值
- **ROUND**：四舍五入到指定小数位数
- **RAND**：生成随机数
- **PI**：返回 π 值
- **SIN**：计算正弦值
- **COS**：计算余弦值
- **TAN**：计算正切值

### 统计函数
- **COUNT**：计算包含数字的单元格数量
- **COUNTA**：计算非空单元格数量
- **COUNTIF**：根据条件计算单元格数量
- **COUNTIFS**：根据多个条件计算单元格数量
- **AVERAGEIF**：根据条件计算平均值
- **AVERAGEIFS**：根据多个条件计算平均值
- **SUMIF**：根据条件计算总和
- **SUMIFS**：根据多个条件计算总和

### 查找和引用函数
- **VLOOKUP**：垂直查找表中的值
- **HLOOKUP**：水平查找表中的值
- **INDEX**：返回数组中指定位置的值
- **MATCH**：返回值在数组中的位置
- **OFFSET**：返回基于指定引用的偏移单元格
- **INDIRECT**：返回由文本字符串指定的引用

### 文本函数
- **CONCAT**：连接多个文本字符串
- **LEFT**：返回文本字符串的左侧字符
- **RIGHT**：返回文本字符串的右侧字符
- **MID**：返回文本字符串的中间字符
- **LEN**：返回文本字符串的长度
- **TRIM**：删除文本字符串中的空格
- **LOWER**：将文本转换为小写
- **UPPER**：将文本转换为大写
- **PROPER**：将文本转换为 Proper 大小写

### 逻辑函数
- **IF**：根据条件返回不同的值
- **AND**：检查多个条件是否都为真
- **OR**：检查多个条件是否至少有一个为真
- **NOT**：反转逻辑值
- **IFERROR**：处理错误值
- **IFNA**：处理 #N/A 错误

### 信息函数
- **ISBLANK**：检查单元格是否为空
- **ISNUMBER**：检查值是否为数字
- **ISTEXT**：检查值是否为文本
- **ISERROR**：检查值是否为错误
- **CELL**：返回有关单元格的信息

## 函数语法和使用方法

### 基本语法
- 函数名称后跟括号
- 括号内包含参数，参数之间用逗号分隔
- 参数可以是常量、单元格引用、表达式或其他函数

### 常见函数示例

#### SUM 函数
**语法**：SUM(number1, [number2], ...)
**说明**：计算指定数值的总和
**示例**：=SUM(A1:A10) - 计算 A1 到 A10 单元格的总和

#### VLOOKUP 函数
**语法**：VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
**说明**：在表格的第一列中查找值，然后返回同一行中指定列的值
**示例**：=VLOOKUP("苹果", A1:B10, 2, FALSE) - 在 A 列查找"苹果"，返回 B 列对应的值

#### IF 函数
**语法**：IF(logical_test, value_if_true, [value_if_false])
**说明**：根据条件返回不同的值
**示例**：=IF(A1>60, "及格", "不及格") - 如果 A1 大于 60，返回"及格"，否则返回"不及格"

#### COUNTIF 函数
**语法**：COUNTIF(range, criteria)
**说明**：计算符合条件的单元格数量
**示例**：=COUNTIF(A1:A10, ">60") - 计算 A1 到 A10 中大于 60 的单元格数量

#### TODAY 函数
**语法**：TODAY()
**说明**：返回当前日期
**示例**：=TODAY() - 返回当前系统日期

#### CONCAT 函数
**语法**：CONCAT(text1, [text2], ...)
**说明**：连接多个文本字符串
**示例**：=CONCAT("Hello", " ", "World") - 返回"Hello World"

## 函数使用的最佳实践

### 1. 正确使用绝对引用和相对引用
- 相对引用：复制公式时会自动调整（如 A1）
- 绝对引用：复制公式时保持不变（如 $A$1）
- 混合引用：行或列保持不变（如 $A1 或 A$1）

### 2. 处理错误值
- 使用 IFERROR 函数捕获错误
- 使用 ISERROR 函数检查错误
- 确保函数参数类型正确

### 3. 优化函数性能
- 避免在大范围数据中使用 volatile 函数（如 RAND、NOW）
- 使用数组公式时注意性能影响
- 适当使用命名范围提高可读性

### 4. 函数嵌套
- 函数可以嵌套使用
- 嵌套层级不宜过深，通常不超过 7 层
- 复杂嵌套可以分解为多个辅助单元格

## 常见错误和解决方法

### 1. #DIV/0! 错误
**原因**：除以零
**解决方法**：使用 IF 函数检查除数是否为零

### 2. #VALUE! 错误
**原因**：函数参数类型错误
**解决方法**：确保参数类型正确，使用适当的函数转换数据类型

### 3. #N/A 错误
**原因**：VLOOKUP 等查找函数未找到匹配值
**解决方法**：使用 IFNA 函数处理，或确保查找值存在

### 4. #REF! 错误
**原因**：引用了无效的单元格
**解决方法**：检查引用的单元格是否存在，避免删除被引用的单元格

### 5. #NAME? 错误
**原因**：函数名称拼写错误或未定义名称
**解决方法**：检查函数名称拼写，确保名称已定义

## 高级函数技巧

### 1. 数组公式
- 按 Ctrl+Shift+Enter 输入数组公式
- 可以同时处理多个值
- 示例：{=SUM(A1:A10*B1:B10)} - 计算两个范围的乘积之和

### 2. 动态数组函数（Excel 365）
- SORT：对数据进行排序
- FILTER：根据条件筛选数据
- UNIQUE：返回唯一值
- XLOOKUP：更强大的查找函数
- SEQUENCE：生成序列

### 3. 日期计算
- DATEDIF：计算两个日期之间的差值
- EOMONTH：返回指定月份的最后一天
- WORKDAY：计算工作日

### 4. 文本处理
- TEXTJOIN：使用分隔符连接文本
- FIND：查找文本在字符串中的位置
- SUBSTITUTE：替换文本中的字符
